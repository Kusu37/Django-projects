# Course Registration

This is a simple Django project for course registration.  
It allows users to view available courses and register for them.

This project is created to practice Django basics such as models, views, URLs, and templates.

---

## Features

- View list of courses  
- Register for a course  
- Simple and easy-to-use interface  

---

## Technologies Used

- Python  
- Django  
- HTML & CSS
